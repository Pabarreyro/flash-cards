Welcome to Your JavaScript Flashcards!

JavaScript Flashcards is a free online resource for learning key JS and jQuery terms.

JavaScript (JS)

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

A special character (or characters) that indicates an action to be performed. Ex: + , - , * , / , %

Variables

Containers used to store information. They allow for a way to label data with a descriptive name.

Variable naming conventions

Use lowerCamelCase when naming JavaScript variables.

Functions

A block of code that performs an action and returns a result; optionally takes arguments.

Methods

An action run on a piece of data; you can think of it as a message you send to a piece of data, and the result is the response.

Arguments

Input values that provide a method with additional information to help it perform its action.

Parameters

A variable within a function to which an argument is passed.

Return

A method's response.

Chaining Methods

Calling a method directly on the return value of another method.

Strings

A JavaScript data type that represents the exact text of whatever is enclosed in the quotes.

Booleans

A data type that denotes true and false states.

Undefined

An object (such as a variable) without a defined value.

NaN

not a number, actually considered a type of number

Escape

Apply // to code that you want the computer to ignore.

Alert

A method that opens a dialog box and returns undefined.

Comments

Anything following // on a line in JavaScript for the purpose of leaving notes in one's code.

jQuery

A fast, small, and feature-rich JavaScript library.

Attributes

Provide additional information to an HTML elements, and can be used to target specific elements and change certain characteristics of objects on a page.

Event Handler

"Listens" to the element(s) to which it's attached and responds when a certain event is triggered.
